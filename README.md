# Scholarum — Simulacro Tienda Escolar

Herramienta para generar simulacros de tiendas escolares.

## Setup

1. Backend: Google Apps Script conectado a Google Sheets (Master DB)
2. Frontend: Esta app React desplegada en Vercel

## Desarrollo local

```bash
npm install
npm run dev
```

## Despliegue

Conectar este repo con Vercel. Se despliega automáticamente en cada push.
